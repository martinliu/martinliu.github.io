安装完美 CentOS7 虚拟机模板

2015-06-05, Friday
阅读时长: 2 分钟

初始化安装

下载最新版 CentOS7 DVD 选择 mini 安装。

网络配置

安装过程中设置了静态网络地址，如下：

[bash]

[root@centos7-tmp ~]# cat /etc/sysconfig/network-scripts/ifcfg-eth0 TYPE=Ethernet BOOTPROTO=none DEFROUTE=yes IPV4_FAILURE_FATAL=no IPV6INIT=no NAME=eth0 DEVICE=eth0 ONBOOT=yes DNS1=192.168.10.1 DOMAIN=xenlab.com IPADDR=192.168.10.8 PREFIX=24 GATEWAY=192.168.10.1 UUID=5fb06bd0-0bb0-7ffb-45f1-d6edd65f3e03

[/bash]

Repo 软件更新源

Change log:

  1. 初始化安装，添加了个几个国内的源

系统服务配置

Change log:

  1. 关闭 SELinux

  2. 关闭 NetworkManager

  3. 关闭 FirewallD

  4. systemctl enable acpid.service

  5. 开启 truned-adm virtual-guest 服务模式

  6. systemctl enable ntpd

[bash]

[root@centos7-tmp tuned]# tuned-adm  list Cannot talk to Tuned daemon via DBus. Available profiles:

  • balanced
  • blog
  • latency-performance
  • network-latency
  • network-throughput
  • powersave
  • throughput-performance
  • virtual-guest
  • virtual-host Cannot talk to Tuned daemon via DBus. It seems that tuned daemon is not running, preset profile is not activated. Preset profile: virtual-guest

[/bash]
