Minikubeдёәеҗ„з§Қж“ҚдҪңзі»з»ҹзҡ„ејҖеҸ‘иҖ…пјҢжҸҗдҫӣдәҶиҝҗиЎҢK8SжңҖз®ҖжҙҒзҡ„ж–№ејҸгҖӮе®ғжңҖе°ҸеҢ–дәҶе®үиЈ…k8sзҡ„йңҖжұӮе’Ңе·ҘдҪңйҮҸгҖӮиҖҢдё”еңЁеҗ„з§Қж“ҚдҪңзі»з»ҹдёҠеҸҜд»Ҙе®һзҺ°зӣёеҗҢзҡ„дҪ“йӘҢгҖӮжң¬ж–Үз”ЁзІҫз®Җзҡ„иҜӯиЁҖпјҢжҸҸиҝ°дәҶд»Һе®үиЈ…minikubeеҲ°иҝҗиЎҢе®№еҷЁеҢ–еә”з”Ёзҡ„иҝҮзЁӢгҖӮдё»иҰҒе‘Ҫд»Өе’Ңиҫ“еҮәжқҘиҮӘmacOSгҖӮ

е®үиЈ…Hypervisor

йңҖиҰҒеңЁд»»дҪ•з¬”и®°жң¬жҲ–иҖ…е·ҘдҪңдёҠе®үиЈ…дёҖдёӘHypervisorпјҢеӣ жӯӨйҰ–е…ҲдҪ йңҖиҰҒеқҡжҢҒз”өи„‘зҡ„CPUжҳҜеҗҰејҖеҗҜдәҶиҷҡжӢҹеҢ–зҡ„ж”ҜжҢҒпјҢжЈҖжҹҘBIOSзҡ„VT-XжҲ–иҖ…AMD-vзҡ„й…ҚзҪ®гҖӮ

  • OS XпјҢе®үиЈ… xhyve driver, VirtualBox жҲ– VMware Fusion
  • LinuxпјҢ е®үиЈ… VirtualBox жҲ– KVM
  • WindowsпјҢ е®үиЈ… VirtualBox or Hyper-V

Minikubeдјҡз”ҹжҲҗдёҖдёӘиҷҡжӢҹжңәпјҢз”ЁдәҺиҝҗиЎҢдёҖдёӘе®үиЈ…е’ҢйғЁзҪІеҘҪзҡ„k8sеҚ•иҠӮзӮ№зі»з»ҹгҖӮ

е®үиЈ…kubectl

kubectlжҳҜk8sзі»з»ҹзҡ„з®ЎзҗҶе·Ҙе…·пјҢжҳҜдёҖдёӘе‘Ҫд»ӨиЎҢе·Ҙе…·пјҢе®ғз”ЁдәҺе’ҢmasterдәӨдә’пјҢе®ҢжҲҗзҫӨйӣҶе’ҢжңҚеҠЎзҡ„з®ЎзҗҶзӯүе·ҘдҪңгҖӮ

kubectlзҡ„е®үиЈ…ж–ҮжЎЈпјҡ https://kubernetes.io/docs/tasks/tools/install-kubectl/

еңЁmacOSдёҠжңҖз®ҖеҚ•зҡ„е®үиЈ…ж–№жі•жҳҜдёӢйқўиҝҷжқЎе‘Ҫд»Өпјҡ

brew install kubectl

brewзҡ„еҘҪеӨ„жҳҜеҗҺз»ӯеҸҜд»Ҙеё®еҝҷжҢҒз»ӯзҡ„еҚҮзә§гҖӮ

е…¶е®ғйқһbrewе®үиЈ…ж–№жі•пјҢи§Ғд»ҘдёҠе®үиЈ…ж–ҮжЎЈзҡ„ Install kubectl binary via curl йғЁеҲҶгҖӮ

е®үиЈ… minikube

еңЁmacOSдёҠдҪҝз”Ёзҡ„й»ҳи®Өзҡ„HypervisorжҳҜVirtualBoxпјҢеҰӮжһңжғіиҰҒз”ЁxhyveйңҖиҰҒеңЁеҗҜеҠЁзҡ„ж—¶еҖҷеҠ дёҠеҸӮж•° --vm-driver=xhyve

е®үиЈ…жңҖж–°зүҲжң¬зҡ„ minikube йңҖиҰҒеҸӮиҖғиҝҷдёӘзҪ‘йЎө https://github.com/kubernetes/minikube/releases
